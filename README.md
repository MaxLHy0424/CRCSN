# 简介

你是否因为无法脱离机房老师的控制, 而无法欢快的使用机房的电脑?

这个软件可以帮到你!

Computer Room Control Software Nemesis (中文名 机房控制软件克星 , 简称 CRCSN), 可以帮助你通过简单的操作快速脱离控制!

# 注意事项

此软件仅可用于 Windows 平台.

此软件仅用于学习交流, 禁止用于商业用途.

此软件按照 MIT 协议开源, 请在遵守 MIT 协议的情况下使用.

如果此软件对计算机造成影响 (包括但不限于: 卡顿, 死机), 请勿使用.
